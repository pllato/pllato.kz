# Google Ads: учёт заявок и решение по оплате

_Обновлено: 08.07.2026. Статус: сайт полностью готов, осталось одно действие в кабинете Google Ads._

## Почему в кабинете «непонятно, были ли обращения»

До 08.07.2026 в Google Ads не было настроено отслеживание конверсий:
тега конверсии (AW-…) не существовало ни на сайте, ни в GTM, страницы
«Спасибо» тоже не было. Кабинет показывал клики и расход, но ноль
конверсий — **не потому, что обращений не было, а потому, что их никто
не считал.**

## Были ли заявки с Google — проверка за 2 минуты

С **25 июня 2026** каждая заявка с рекламного клика Google содержит
в Telegram-сообщении строку `gclid: …`.

1. Открыть группу **«Заявки с сайта pllato.kz и других источников»**.
2. Поиск по чату → ввести `gclid`.
3. Количество найденных сообщений = заявки с рекламы Google с 25 июня.

Дополнительно: поиск `Источник: https://www.google` покажет заявки из
Google вообще (реклама + SEO, за всё время). Заявки до 25 июня по
источнику достоверно не атрибутируются.

## Критерий решения по платежу

- **Клики были, заявок с `gclid` ноль** → на паузу, платёж не вносить.
  Сначала перенастроить: минус-слова, целевые запросы, посадочная
  `/crm/svoya-crm/`, конверсии по `/thanks/` (см. ниже). Без учёта
  конверсий деньги уходят вслепую.
- **Заявки есть** → цена заявки = расход ÷ число заявок с gclid.
  При чеке заказной CRM (миллионы тенге) одна сделка окупает месяцы
  рекламы — тогда платить и масштабировать то, что работает.

## Что уже готово на сайте (ничего делать не нужно)

- `/thanks/` — страница после успешной отправки любой формы;
  редирект работает на всех лендингах и главной.
- Событие `thank_you_view` в dataLayer на этой странице,
  `lead_submitted` — в момент отправки формы.
- `gclid`/`gbraid`/`wbraid`/`fbclid` + все UTM (включая `utm_content`)
  приходят в каждой Telegram-заявке, cookie живёт 90 дней.
- GTM (GTM-M5N2WR2B) и Meta Pixel — на 100% публичных страниц.

## Как включить конверсии — два пути

### Путь А — быстрый, без GTM (рекомендую)

1. Google Ads → **Цели → Конверсии → + Новое действие-конверсия**.
2. Тип **«Веб-сайт»**, ввести `pllato.kz`, дальше **«Создать вручную»**.
3. Категория: **«Отправка формы для потенциальных клиентов»**,
   название: `Заявка с сайта (thanks)`, значение — не указывать,
   учёт: **«Одна конверсия»** на клик.
4. На шаге установки тега выбрать **«Установить тег самостоятельно»**
   и прислать Claude две строки из кода: `AW-XXXXXXXXXX` и
   `метку конверсии` (conversion label, строка после слэша в
   `send_to`).
5. Claude сам вшивает тег на `/thanks/` и выкатывает в прод —
   конверсии начинают считаться в тот же день.

### Путь Б — через GTM (для таргетолога, если он ведёт контейнер)

В контейнере **GTM-M5N2WR2B**:

1. Тег **Conversion Linker**, триггер All Pages (если ещё нет).
2. Тег **Google Ads Conversion Tracking**: Conversion ID и Label из
   кабинета (см. Путь А, шаги 1–3).
3. Триггер: **Custom Event** `thank_you_view`
   (или Page View, Page Path равен `/thanks/`).
4. Опубликовать контейнер, проверить в режиме Preview: отправить
   тестовую заявку с номером `+7 701 123 99 99` (владельческий номер —
   уведомление не уходит, но редирект на `/thanks/` происходит).

### Проверка после настройки

Google Ads → Цели → Конверсии: статус должен смениться на
«Недавних конверсий нет» → после первой заявки на «Активна»
(данные появляются с задержкой до 24 часов).

## Сообщение таргетологу (скопировать и отправить)

> На сайте готова конверсионная страница pllato.kz/thanks/ — туда
> редиректит после успешной отправки любой формы. В dataLayer есть
> события thank_you_view (на /thanks/) и lead_submitted (в момент
> отправки). Настрой, пожалуйста, в Google Ads действие-конверсию
> «Отправка формы» и повесь её через GTM (GTM-M5N2WR2B): Conversion
> Linker + Google Ads Conversion Tag на событие thank_you_view.
> Если проще без GTM — создай конверсию вручную и скинь мне
> Conversion ID (AW-…) и Label, тег вошьют в код страницы.
> В Telegram-заявках уже приходят gclid и все UTM, включая
> utm_content, — можно сверять кабинет с фактическими лидами.

## Дальше (когда конверсии заработают)

1. Вести рекламу на `/crm/svoya-crm/` и профильные лендинги
   (`/crm/na-zakaz/`, `/crm/vnedrenie/`), а не на главную.
2. Через 2–3 недели включить в кампаниях стратегию
   «Максимум конверсий» вместо кликов.
3. Раз в неделю: отчёт «Поисковые запросы» → нецелевые в минус-слова.
