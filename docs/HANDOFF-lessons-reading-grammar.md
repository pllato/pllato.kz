## Что это за система
Пробный урок (trial lesson) для студентов ELC. Шаги: Уровень → Тур → Слова →
Грамматика → **Чтение** → Спикинг → Регистрация.

- **Студент** проходит урок на своём телефоне на `https://elc-lesson-worker.uurraa.workers.dev/lesson?id=...`
- **Учитель** видит материал и жмёт Зачёт/Риворк в портале `team.html` → вкладка «Пробные Занятия» → «Проверка» (`renderLessonChecks`).
- **Контент** уроков (грамматика, чтение, слова, спикинг) — файл `lessons.js`, который отдаёт воркер по `/lessons.js`, а `team.html` грузит как `window.ELC_LESSONS`.

**Где что лежит:**
| Что | Файл / место | Репозиторий |
|---|---|---|
| Контент уроков (grammar/reading/…) | `lessons.js` | `pllato/elc-lessons` |
| Страница урока студента `/lesson` + сессия | код воркера (index/worker) | `pllato/elc-lessons` |
| Карточка учителя «Проверка» | `team.html` → `renderLessonChecks` (~строка 4386) | `pllato/pllato.kz` |

## Фидбек преподавателей (что чинить)
**Задача 3 — Чтение (главная боль).**
Симптом: текст чтения показывается только у одного устройства; когда открывается у второго — «исчезает у первого». Плюс текст виден только у учителя после кнопки Ready — приходится снимать телефоны студентов, неэффективно.
Диагноз (гипотеза): текст привязан к одному общему состоянию сессии и «перехватывается» тем, кто открыл последним; не сохраняется.
**Одобренное решение:**
- Текст чтения генерится **один раз на занятие и сохраняется в сессии** (в БД/состоянии воркера).
- Показывается **сразу с начала задания** и **одновременно у студента и у учителя** (обе стороны читают одну сохранённую копию) → баг «исчезает у второго» уходит, телефоны снимать не нужно.
- Правка — в коде страницы `/lesson` воркера (`pllato/elc-lessons`), НЕ в `lessons.js`.

**Задача 4 — Грамматика.**
Симптом: 1 пример на правило; 3 темы уровня идут разрозненно.
**Одобренное решение (согласовано с админом):**
- **5 примеров** на каждый уровень.
- **Мини-таблица-шпаргалка** (Правило / Формула / Пример) — связывает 3 темы в один мини-урок.
- Формат подтверждён: «по 5, мини-табличка, видно сразу и всем».

## Модель данных grammar (в lessons.js) — новый формат
```js
grammar: {
  title: '…',
  form: '…',
  table: [                         // НОВОЕ: мини-таблица-шпаргалка
    { rule:'Present Perfect', form:'have/has + V3', example:'I have met a new friend.' },
    { rule:'1st conditional', form:'If + Present, will + V', example:'If you smile, you will find friends.' },
    { rule:'Quantifiers', form:'much / many / a lot of', example:'We have a lot in common.' },
  ],
  examples: ['…', '…', '…', '…', '…'],  // 5 штук
  test: [ … ],                     // без изменений
}
```
