# CHANGELOG

История ревизий Pllato CRM. Свежие сверху.

См. также `app/version.js` (читается в UI → Настройки → О приложении).

---

## v0.8 · rev-7 (2026-05-13)
- Кнопки коммуникации (📞 / 💬 / ✉) в карточке сделки и контакта.
- Модалка `app/communicate.js` с выбором канала из доступных (по типу).
- Активность пишется в `deal_activities` / `contact_activities`.
- Реальная отправка — заглушка alert (будет через Worker).

## v0.7 · rev-6 (2026-05-13)
- **Контакт-центр** в `pllato.kz/contact-center.html` — админка каналов связи (Binotel, Green-API WA, SMTP, Instagram, Facebook).
- `/channels` (public) + `/channel_secrets` (private) в Firebase RTDB.
- Read-only блок «Каналы связи» в Настройках CRM, читает из Firebase.
- `app/channels.js` — кэширование + sync.

## v0.6 · rev-5 (2026-05-13)
- Единая база сотрудников из Firebase `/users` (общая для всех приложений Pllato).
- При входе в CRM подтягивается полный список + миграция старых `assigneeId`/`participantIds`/`authorId` по email.
- Раздел «Сотрудники» в Настройках стал read-only (управление в `pllato.kz/app.html`).
- Проверка доступа `apps.pllato_crm` при входе в CRM.

## v0.5 · rev-4 (2026-05-13)
- **Google Sign-In** в `/crm/` через Firebase Auth (тот же проект `pllato-crm`).
- Проверка `/users` для авторизации в команду.
- Searchable селекты «Контакт» и «Ответственный» в форме сделки.
- Quick-create контакта прямо из выпадающего списка.
- Умный импорт контактов (CSV / текст / Excel paste): автодетект колонок, опция «создавать сделки».

## v0.4 · rev-3 (2026-05-13)
- Переименование «Pllato CORE CRM» → «Pllato CRM» во всех UI-строках.
- Деплой на `pllato.kz/crm/` через GitHub Pages.
- Подключён общий Firebase `pllato-crm`.

## v0.3 · rev-2 (2026-05-13)
- Роли фильтруют меню по правам доступа.
- Custom fields в сделках (настройка + применение в форме).
- Сохранение позиции / состояния при reload (sessionStorage).
