🔗 Генератор временных ссылок

Ссылка работает до указанной даты, потом клиент видит «Срок истёк». Страница-генератор непубличная — не давайте её клиентам.

Как это работает. Список берётся с портала pllato.kz/app автоматически: каждая плашка и все документы из её выпадашки. Новая плашка на портале сразу появится и здесь. Срок ссылки проверяет гейт на самой странице: без свежего токена клиент попадает на «Срок истёк». Вход в портал автоматически открывает все такие страницы на вашем устройстве — ссылки нужны только клиентам.

Чтобы новая страница реально закрывалась по сроку: подключите первым тегом в <head> скрипт <script src="/app/gate.js"></script>. Для страницы без гейта ссылка тоже сгенерируется, но срок не будет ограничен — генератор предупредит.
Для себя: открыть вечный доступ на этом устройстве
