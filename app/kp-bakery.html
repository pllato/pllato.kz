📱 Торговый агент — мобильное приложение

  • Заявки Т-2/Т-1: календарь 30 дней, потоварные дедлайны ±, автоподстановка, статусы модерации
  • Умная корзина: Блок А/Б, сетевые матрицы, сканер штрихкодов, весовой товар
  • 5 сценариев оплаты, FIFO-гашение ДЗ, защита лимитов
  • Перегруз в полях, расходы, KPI и зарплатная карточка
  • Bluetooth-печать чеков (ESC/POS, 58/80 мм)
  • Offline-first: SQLite + шифрование, GUID-идемпотентность

🏭 Кладовщик СГП — веб (адаптив)

  • Приёмка с производства по датам выработки, конвертация шт→кг
  • Сводная потребность + мульти-суммирование агентов
  • Модерация корректировок с push 3 уровней
  • Отгрузка с авто-типом накладной, выравнивание остатков ВСА
  • Закрытие смен: пересчёт, расчёт недостач/излишков
  • Касса СГП: розница + продажа персоналу под ЗП

👁️ Супервизор — веб

  • GPS-карта: треки (точка / 5 мин), маркеры визитов, километраж
  • Фотоконтроль с гео-метками
  • ДЗ команды, разовые разблокировки консигнации
  • Модерация графиков до 12:00 Т-1
  • Аудит расходов, звонков, чатов своей команды

⚙️ Администратор — Back-Office

  • Каталог SKU: штрихкоды, весовой флаг, кванты, сроки
  • Иерархия прайсов 3 уровней + матрицы сетей
  • Отчёт выработки цеха (изолирован от продаж)
  • Зарплатный отчёт: раздельные регистры А/Б
  • Маржинальность, «уснувшие» ТТ, анти-фрод по возвратам
  • Телефония, WhatsApp-рассылки, распределение банкетных заказов

Почему это дороже типовой CRM: техническая нагрузка

Честно показываем, что именно в вашем ТЗ создаёт дополнительную сложность — и учтено в цене.

📵 Offline-first архитектура

Полноценная локальная БД (SQLite + SQLCipher) на телефоне, очередь синхронизации, GUID-идемпотентность против задвоения выручки. Это фактически вторая база данных со своей логикой конфликтов — ×1.5 к сложности мобильной части.

📍 Фоновый GPS-трекинг

Foreground Service, работа при заблокированном экране, анти-дрейф фильтр, оффлайн-буфер точек. Отладка на «капризных» прошивках (Xiaomi, Huawei) — отдельный цикл тестирования на реальных устройствах.

🖨️ Bluetooth-печать ESC/POS

Прямая работа с термопринтерами 58/80 мм, адаптивная вёрстка чека под две ленты, тестирование на физических принтерах.

💬 WhatsApp Business API + IP-телефония

Верификация Meta (1–3 недели), шлюз маршрутизации чатов по номеру ЛПР, интеграция с облачной АТС, запись звонков с привязкой к ТТ.

🖥️ Поддержка Windows XP/7 на складе

Отдельная облегчённая фронт-сборка без современных JS-фич: пагинация, virtual scrolling, никаких тяжёлых библиотек.

🧮 Сложная финансовая логика

5 сценариев оплаты × 4 варианта консигнации, FIFO-гашение, виртуальные авансы, изолированные зарплатные регистры А/Б, многодневные рейсы — каждая ветка покрывается тестами, ошибки здесь = прямые деньги клиента.

Ядро: маршрут работает Недели 1–8

  • Мобильное приложение агента: заявки Т-2/Т-1, корзина А/Б, 5 сценариев, ДЗ + FIFO, offline-синхронизация
  • Веб кладовщика: приёмка, модерация корректировок, отгрузка + ВСА, закрытие смен
