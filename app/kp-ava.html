Коммерческое предложение для ОсОО «Ава Тават» (Бишкек) — единственного дистрибьютора проф-косметики Concept по Кыргызстану (~700 наименований, салоны/магазины/частные). Уходим от неудобного SalesDoctor к своей системе под ваши процессы: удобный товар с папками и ценами, воронка, история клиентов, возвраты по сканеру, прибыль дня и мобильный интерфейс агента. Полное владение, на вашем сервере.

Что входит в разработку
1Товары, папки и 3 цены
Категории-папки, 3 цены в строке (закуп/опт/розница), массовая смена цен папки, карточка товара (остаток, маржа, мин. запас), автоподгрузка прайса.
2Возвраты по сканеру
Возврат по клиенту: сканер → на склад + в историю. Корректировка/ревизия/инвентаризация склада.
3Клиенты и история
Салоны/магазины/частные, статусы, история покупок и отказов, блокировка проблемных клиентов.
4Воронка продаж
Обращения из Instagram/WhatsApp → воронка по статусам; сбор всех данных в одном месте.
5Прибыль дня + закуп
Калькулятор прибыли/убытка за день по марже и расходам; рекомендации по закупу на основе истории продаж.
6Триггеры
Авто-события: день рождения, повторный заказ, «вернувшийся клиент», низкий остаток — напоминания и рассылки по статусу.
7Мобильный интерфейс агента
Упрощённое приложение: агент в пару касаний шлёт заявку в офис, офис её формирует.
8Экспедиторы и рейсы
Свои/наёмные машины, рейсы, вечерний возврат отказов. Каталог машин внутри процесса.
9AI-справочник
Бот из Instagram/WhatsApp: 2–3 вопроса по каталогу → рекомендация линейки → перевод на менеджера. Бесплатные модели (DeepSeek), токены по вашему ключу.
10Дашборд + роли
Обзор с прибылью дня, продажами, вниманием; роли (руководитель, менеджер, товаровед, агент, экспедитор, маркетолог).
