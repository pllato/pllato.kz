# ELC World — демо 3D-мира школы (в стиле Roblox)

Демо-версия веб-игры для школы английского языка ELC: блочный 3D-мир,
в котором дети гуляют по кампусу школы, говорят с учителями, выполняют
квесты и собирают монеты. Визуальный стиль — как в Roblox (блочные
аватары, яркие цвета, таблички с именами), но это полностью своя игра.

**Live (после merge):** `https://pllato.kz/elc-world/`

## Что уже есть в демо

- Блочный аватар (R6-стиль) с анимацией ходьбы/прыжка и смайлом
- Карта кампуса: здание школы с классом (доска, парты), офис с
  расписанием и «ELC CRM» на компьютере, детская площадка, паркур (обби)
- Камера от третьего лица: мышь/тач-drag, зум колесом, прижатие у стен
- Квест «Собери буквы HELLO» (мисс Анна) → награда монетами
- Английский квиз с вариантами ответов (мисс Эмма) → награда
- NPC-«игроки» (Timur, Aisha), которые гуляют и болтают
- Монеты, конфетти, звуки (WebAudio, без файлов)
- HUD в стиле Roblox: счётчик монет, список игроков, трекер заданий,
  диалоги с кнопками, загрузочный экран
- Мобильное управление: виртуальный джойстик, кнопки прыжка и «💬»

## Технически

- Один файл `index.html`: vanilla JS + three.js r160 (модуль лежит рядом
  в `lib/three.module.min.js`, без CDN и без сборки — как весь сайт)
- Вся карта, текстуры (canvas) и звуки генерируются кодом — ассетов нет
- Физика: простые AABB-коллайдеры (стены, платформы, потолки)
- Отладочные хуки для тестов: `window.__game` (teleport, state, …)

## Как развивать дальше (идеи)

- Редактор карты / загрузка карты из JSON (Firebase RTDB)
- Реальный мультиплеер (WebSocket / Cloudflare Workers + Durable Objects)
- Аккаунты учеников: монеты и прогресс в Firebase, привязка к CRM
- Задания от учителей из CRM → квесты в мире; магазин наград
- Кастомизация аватара (цвета, одежда, питомцы)

## Локальный запуск

```bash
python3 -m http.server 8080
# открыть http://localhost:8080/elc-world/
```
