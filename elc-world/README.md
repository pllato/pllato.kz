# ELC World — 3D-мир школы английского (демо 0.2)

Демо веб-игры для школы английского языка ELC: компактный, но детально
проработанный блочный 3D-мир в стиле топовых Roblox-игр. Дети гуляют по
кампусу, говорят с учителями, выполняют квесты и собирают монеты.
Это полностью своя игра (логотипы/названия Roblox не используются).

**Live:** `https://pllato.kz/elc-world/`

## Графика (v0.2 — «дорогая» картинка)

- Скруглённые блоки (RoundedBoxGeometry) вместо острых кубов
- ACES-тонемаппинг, мягкие тени, окружение (PMREM RoomEnvironment)
- Bloom-постобработка: неоновые градиентные вывески («ELC WORLD»,
  «ELC SCHOOL»), светящиеся фонари, солнце — как в Pls Donate
- Градиентное небо (шейдер), пушистые облака из сфер
- Фонтан с анимированной водой и каплями, сакура с падающими лепестками,
  воздушные шарики на входной арке
- Аватары с причёсками, лицами с бликами, бейджем ELC на футболке
- Детали: маркиза над входом, цветочные ящики на окнах, клумбы, скамейки,
  живая изгородь, паркет и постеры внутри школы

## Мир и геймплей

- Компактная карта: площадь с фонтаном, школа (класс + офис внутри),
  детская площадка, паркур (обби), спавн с аркой
- Квест «Собери буквы HELLO» (мисс Анна) → 50 монет
- Английский квиз (мисс Эмма, 3 вопроса) → 30 монет
- Офис: Mr. Pllato, доска расписания, компьютер «ELC CRM» с расписанием
- NPC-«игроки» Timur и Aisha гуляют и болтают
- HUD: монеты, список игроков, трекер заданий, диалоги, загрузочный экран
- Управление: WASD/мышь на десктопе, джойстик + кнопки на телефоне

## Технически

- Один файл `index.html`: vanilla JS + three.js r160, без сборки
- three.js и аддоны (RoundedBox, EffectComposer, UnrealBloom, OutputPass,
  RoomEnvironment) вендорятся в `lib/` — никаких CDN и npm
- Все текстуры — canvas-генерация, ассетов нет
- Физика: AABB-коллайдеры (стены, платформы, потолки), камера с
  «прижатием» при препятствиях
- Отладочные хуки для тестов: `window.__game`

## Как развивать дальше

- Реальный мультиплеер (Cloudflare Workers + Durable Objects / WebSocket)
- Аккаунты учеников: монеты и прогресс в Firebase, привязка к CRM
- Задания от учителей из CRM → квесты в мире; магазин наград и питомцев
- Кастомизация аватара, новые локации (кафе, стадион, библиотека)

## Локальный запуск

```bash
python3 -m http.server 8080
# открыть http://localhost:8080/elc-world/
```
