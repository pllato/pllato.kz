# pllato.kz

Корпоративный сайт **Pllato** — консалтинг по операционной отцифровке бизнеса (девелопмент, опт, производство, клиники).

## Стек

- Чистый HTML/CSS/JS, без сборщиков и фреймворков
- Все стили инлайн в `<style>` внутри каждого `.html`
- Inter с Google Fonts
- Иконки/лого — SVG в корне

## Страницы

| Файл | Назначение |
|---|---|
| `index.html` | Главная — отцифровка операционки бизнеса |
| `audit.html` | Аудит бизнеса |
| `assessment.html` | Ассесмент (Pllato CRM) |
| `crm-builder.html` | Визуальный конструктор CRM |
| `media.html` | Раздел для медиа |
| `migrate.html` | Мастер миграции (Pllato CRM) |
| `team.html` | Команда |
| `privacy.html` | Политика конфиденциальности |
| `terms.html` | Пользовательское соглашение |

## Деплой

GitHub Pages с кастомным доменом — `CNAME` в корне = `pllato.kz`.

```bash
git add -A && git commit -m "..." && git push origin main
```

После пуша GitHub Pages пересобирает сайт за ~30-60 сек, доступен на https://pllato.kz.

## Замечания

- В корне лежит `pllato-crm_README.md` — это README **другого** проекта (Pllato CRM с Cloudflare Worker + Firebase), а не pllato.kz. Не путать. Если пилим этот лендинг — README трогать не надо.
- Дизайн-система в `:root` верхней страницы (`index.html`): фирменные цвета `--navy: #0a1628`, `--bronze: #b8895a`, плюс акцентные цвета по нишам (недвижимость / опт / производство / клиники).
